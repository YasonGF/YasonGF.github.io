在ubuntu 20.04系统上搭建github+hexo 静态博客

基本环境

ubuntu 20.04

apt 2.0.9 (amd64)

nodejs v18.18.2

npm 9.8.1
hexo-cli@4.3.1

git 2.25.1

项目node_modules
> hexo-site@0.0.0 /codes/blog-gexo/blog
├── hexo-deployer-git@4.0.0
├── hexo-generator-archive@2.0.0
├── hexo-generator-category@2.0.0
├── hexo-generator-index@3.0.0
├── hexo-generator-tag@2.0.0
├── hexo-renderer-ejs@2.0.0
├── hexo-renderer-marked@6.1.1
├── hexo-renderer-stylus@3.0.0
├── hexo-server@3.0.0
├── hexo-theme-landscape@1.0.0
└── hexo@6.3.0

安装git、nodejs、npm

git系统自带，一般都提前安装好了
安装nodejs， 这里可以使用apt包管理器来安装，但是版本(v10.19.0)较为老旧，后面安装hexo可能有依赖问题
这里我们使用Node.js DEB repository来安装

sudo apt-get update && sudo apt-get install -y ca-certificates curl gnupg
curl -fsSL https://deb.nodesource.com/gpgkey/nodesource-repo.gpg.key | sudo gpg --dearmor -o /etc/apt/keyrings/nodesource.gpg
NODE_MAJOR=18
echo "deb [signed-by=/etc/apt/keyrings/nodesource.gpg] https://deb.nodesource.com/node_$NODE_MAJOR.x nodistro main" | sudo tee /etc/apt/sources.list.d/nodesource.list
sudo apt-get update && sudo apt-get install nodejs -y

以上命令会把nodejs环境和npm包管理器都会安装好
通过node -v 和 npm -v 可以查看当前安装的版本

配置github

设置git 用户信息
git config --global user.name "GitHub 用户名" git config --global user.email "GitHub 邮箱"

配置ssh公钥
ssh-keygen -t rsa -C 'your github email'
一路回车，将生成的ssh公钥配置到github的setting界面，选择ssh and gpg kyes， 添加新的ssh-key
添加成功的验证方法： ssh -T git@github.com

设置github person access token
在github的setting界面选择developer setting， 添加TOKEN. git提交时需要验证PAT

新建github repository
仓库名称： github用户名.github.io

hexo

安装hexo-cli：sudo npm install -g hexo-cli

初始化hexo：hexo init my-blog

cd my-blog

下载项目相关依赖
npm install

生成界面
hexo g

启动预览
hexo s

访问链接： http://localhost:4000 or http://ip_addr:4000

hexo项目结构
tree -L 1

├── _config.landscape.yml
├── _config.yml
├── db.json
├── LICENSE
├── node_modules
├── package.json
├── package-lock.json
├── public
├── README.md
├── scaffolds
├── source
└── themes

_config.yml 网站配置信息
package.json 应用程序信息
scaffolds 模板文件夹
source 存放用户资源，即.md文档
themes 主题文件夹
public 网站文件

部署hexo到github pages

安装一键部署插件
npm install hexo-deployer-git --save

修改_config.yml文件

deploy:
  type: git
  repository: git@github.com:用户名/用户名.github.io.git
  branch: main

部署步骤
hexo g && hexo d
