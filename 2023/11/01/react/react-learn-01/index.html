react 项目结构

node_modules: 存放项目依赖的包
public:
src: 项目源代码

src下的App.js是该项目的根组件
src下的index.js是整个项目的入口，负责导入react的核心包和根组件App，并将App根组件渲染到public/index.html中的一个id为root的dom节点上

package.json: 管理依赖库的版本， 类型于maven中的pom.xml文件

基本介绍

JSX (Javascript XML)： 一种JavaScript的语法扩展（浏览器本身不能识别，需要解析之后才能在浏览器中运行），
可以在JavaScript代码中编写HTML模板结构，也是React中编写UI模板的方式。

优势：HTML的声明式模板写法，JavaScript的可编程能力

babel js

jsx 使用js表达式

在jsx中可以通过{}识别js中的表达式，比如变量、函数调用、方法调用…

使用引号传递字符串

 {'this is message'}

使用js变量

const COUNT = 100

function App() {
  return (
    <div className="App">
      {COUNT}
    </div>
  );
}

函数调用和方法调用


function foo() {
  return 'foo bar'
}

function App() {
  return (
    <div className="App">

      {foo()}
      <br />
      {new Date().getDate}
    </div>
  );
}

使用js对象

function App() {
  return (
    <div className="App">
      <div style={{ color: 'red' }}>this is div</div>
    </div>
  );
}

attention： if语句、switch语句、变量声明属于语句，不是表达式，不能出现在{}中

jsx使用列表渲染

const list = [
  { id: 1001, name: 'Vue' },
  { id: 1002, name: 'React' },
  { id: 1003, name: 'Angular' }
]

function App() {
  return (
    <div className="App">
      this is app

      </ul>

    </div>
  );
}

jsx实现复杂条件渲染

需求： 列表中需要根据文章状态设配3中情况，单图、三图、无图三种模式

const articleType = 1   // 0 1 3

function getArticleType() {
  if (articleType == 0) {
    return <div>我是无图的</div>
  } else if (articleType == 1) {
    return <div>我是单图的</div>
  } else if (articleType == 3) {
    return <div>我是三图的</div>
  }
}

function App() {
  return (
    <div className="App">
      {getArticleType()}

    </div>
  );
}

React事件

React基础事件绑定

on + 事件名称 = {事件处理程序}， 遵循驼峰命名

function App() {
  const handleClick = () => {
    console.log('button被点击了...')
  }

  return (
    <div className="App">
      this is app <br />
      <button onClick={handleClick}>click me</button>
    </div>
  );
}

使用事件对象参数

  const handleClick = (e) => {
    console.log('button被点击了...', e)
  }

传递自定义参数

事件绑定的位置改成箭头函数的写法

function App() {
  const handleClick = (e) => {
    console.log('button被点击了...', e)
  }

  return (
    <div className="App">
      this is app <br />
      {/**/}
      <button onClick={() => { handleClick('jackson') }}>click me</button>
    </div>
  );
}
